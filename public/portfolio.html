Project One — Complex Existing Server Optimization

Problem This customer had a previous MSP that was unable to deliver on a server upgrade project and this company had server infrastructure that was in disarray. 2012 servers in use.

Solution Rebuild the entire server infrastructure and failover cluster from scratch ensuring zero downtime and full use of existing storage and hardware. Decommission 2012 servers and virtualize their workloads.

Result I created all the diagrams used for planning that were presented to the business owner.

  • Virtual Infrastructure Plan included all of the existing hardware and storage and the most efficient way I found to consolidate everything.
  • Storage Plan includes the existing storage volumes and how I was going to organize them.
  • Server Licensing broke down the existing licensing (uncompliant) and the new licensing requirement for Windows Server 2022.
  • Final Diagram is the final result. We were able to execute this plan with one late night onsite visit and zero downtime for the customer.

Project Two — Infrastructure Optimization and Failover Setup

Problem This new customer was not fully utilizing their server infrastructure to reduce costs and improve workflow efficiency. They also had previously purchased a failover server for a stretch cluster that was never deployed.

Solution Migrate from using JumpCloud logins to unifying their workstations with a site-to-site VPN to ensure AD domain access. Then move their data and workloads to the existing RDS infrastructure. Followed by locking local PCs down with group policies and basic images. Finally, configure stretch failover cluster between two sites.

Result An efficient AD environment with a stretch failover cluster for disaster recovery.

  • I created the plans that were presented to the stakeholders and worked with onsite IT personnel to execute on the changes for unifying the domain.
  • I identified the existing server environment and how to fit it within a smaller physical server in another city. I came up with the plan and executed deployment of server including onsite install.
